# Setup dependency

>**Clone my repo**
```
git clone https://github.com/narmyen/unit-test-sorted-array.git
```

>**npm init**
```
npm init -y
```

>**ติดตั้ง TypeScript และ Jest**
```
npm install --save-dev typescript ts-jest jest @types/jest
```

>**คอมไพล์ TypeScript**
```
npx tsc
```

>**รันการทดสอบด้วย Jest**
```
npx jest
```
